The forces
The thin bar under each slider = active leverage: this force's current exposure × local sensitivity. It is not importance; a low bar can still hide a meaningful first nudge. Drag one and watch the others change — that interaction is what complexity is.

boundaryCost        = (async + state + 0.5) × (1.5 − opMaturity)
stayingTogetherCost = changeRate × coupling × (0.5 + complexity)
failureSurface      = if (async>0.6 && state>0.6 && opMaturity<0.4)
                          → high plateau   (this is the snap)
                      else → low, gentle
decisionPressure    = (blastRadius + irreversibility) / 2
                      // low → cheap & reversible → delay; high → decide deliberately

structuralNovelty = 0.45×coordinationNovelty + 0.30×complexity + 0.25×(1−familiarity)
agentTrust        = familiarity × (1 − blastRadius) × (1 − structuralNovelty)
                    // green where it's well-trodden & cheap to be wrong
observabilityNeed = async × (0.5 + 0.5×state) × (0.5 + 0.3×coupling + 0.2×deploymentScale)
                    // deployment scale buys its own observability bill
observabilityGap  = observabilityNeed − opMaturity     // red when the team can't pay it
debuggingPain     = asyncKnee(async) + 0.4×coupling    // knee at async 0.5; clamps at 1
cognitiveLoad     = (coupling + 0.25×coordinationNovelty) × (0.4 + 0.6×async) × (0.5 + 0.5×state)

safetyPenalty     = 0.18×(1−opMaturity)×async + 0.12×blastRadius×state
                    + 0.20×max(0, observabilityGap) + 0.10×failureSurface
                    // a thin team can't safely run a hard boundary — this pushes
                    // the region lines toward "keep together". The map moves.

field x = stayingTogetherCost      field y = boundaryCost
region  = sign(x − y − safetyPenalty); not-ripe band width ∝ (1 − decisionPressure)
so opMaturity & blastRadius move the region lines, not just the dot.

                    sensitivity, capped to a stable visual scale. Low when a force
                    is almost absent; a first nudge can still matter.
